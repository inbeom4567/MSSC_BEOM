# Plan: 그래프 자동 생성 (graph-generation)

## 1. 개요

수학 문제 풀이에서 그래프가 필요한 경우, Claude가 Matplotlib 코드를 생성하고 백엔드에서 실행하여 그래프 이미지를 결과에 포함시킨다.

## 2. 요구사항

### 필수
- Claude가 풀이 중 그래프가 필요한 부분을 자동 감지
- Matplotlib으로 정확한 수학 그래프 생성 (좌표, 눈금, 라벨 정확)
- 원본 해설의 그래프 스타일을 최대한 따름
- 생성된 그래프 이미지를 풀이 결과에 포함하여 표시
- 이차함수, 삼차함수, 삼각함수, 로그함수, 지수함수 등 고등학교 수준 그래프 지원

### 선택
- 그래프 스타일 옵션 (격자 표시, 색상, 축 범위)
- 여러 함수를 하나의 그래프에 겹쳐 그리기
- 교점, 극값, 점근선 등 주요 포인트 표시

## 3. 구현 전략

### 3.1 Claude 출력 형식

Claude가 그래프가 필요한 곳에 특수 태그를 삽입:
```
-그래프-
함수: y = x^2 - 4x + 3
x범위: -1, 5
포인트: (1, 0), (3, 0), (2, -1)
라벨: y = x² - 4x + 3
-그래프끝-
```

### 3.2 백엔드 처리 흐름

```
Claude 응답 수신
  → -그래프- 태그 파싱
  → Matplotlib 코드 생성 + 실행
  → PNG 이미지 생성 (base64)
  → 태그를 이미지 데이터로 교체
  → 프론트엔드에 전달
```

### 3.3 프론트엔드 표시

- `SolutionDisplay.jsx`에서 그래프 이미지를 감지하여 `<img>` 태그로 렌더링
- 그래프 이미지 다운로드/복사 버튼 제공

## 4. 기술 스택

| 구성요소 | 기술 |
|---------|------|
| 그래프 생성 | Matplotlib (Python) |
| 이미지 형식 | PNG → base64 인코딩 |
| 폰트 | 한글 지원 폰트 (맑은 고딕 또는 나눔고딕) |
| Docker | matplotlib 패키지 추가 필요 |

## 5. 변경 파일

### 백엔드
| 파일 | 변경 내용 |
|------|----------|
| `services/graph_service.py` | 신규 - 그래프 태그 파싱 + Matplotlib 실행 |
| `services/claude_service.py` | 수정 - 응답에서 그래프 태그 후처리 |
| `prompts/solve_prompt.txt` | 수정 - 그래프 태그 출력 규칙 추가 |

| `Dockerfile` | 수정 - 한글 폰트 설치 |

### 프론트엔드
| 파일 | 변경 내용 |
|------|----------|
| `components/SolutionDisplay.jsx` | 수정 - 그래프 이미지 렌더링 |
| `components/GraphImage.jsx` | 신규 - 그래프 이미지 표시/다운로드 컴포넌트 |

## 6. 구현 순서

1. **matplotlib 설치 + graph_service.py 생성** - 그래프 태그 파싱 및 이미지 생성
2. **프롬프트 수정** - Claude에게 그래프 태그 출력 규칙 추가
3. **claude_service.py 수정** - 응답 후처리에서 그래프 생성 연동
4. **프론트엔드 수정** - 그래프 이미지 렌더링 + 다운로드
5. **Docker 업데이트** - matplotlib + 한글 폰트 설치
6. **테스트** - 이차함수, 삼각함수 등 다양한 그래프 검증

## 7. 리스크

| 리스크 | 대응 |
|--------|------|
| Docker 내 한글 폰트 없음 | Dockerfile에 나눔고딕 설치 추가 |
| Matplotlib 렌더링 시간 | 간단한 그래프는 1초 미만, 복잡해도 3초 이내 |
| Claude가 태그 형식을 안 지킴 | 프롬프트에 예시를 충분히 포함 |
| 그래프 좌표 범위 부적절 | Claude에게 적절한 범위 지정 규칙 추가 |

## 8. 성공 기준

- 이차함수, 삼각함수, 로그함수 그래프가 정확하게 생성됨
- 원본 해설의 그래프와 유사한 스타일
- 풀이 결과에 그래프가 자연스럽게 포함되어 표시됨
- Docker 환경에서도 한글 라벨이 정상 표시됨
