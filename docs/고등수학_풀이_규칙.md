### 1-3. 비격식 표현 → 격식 표현

| 비격식 | 권장 |
|---|---|
| ~로 두자 | ~라 하자 |
| ~를 만들면 | ~을 세우면 |
| ~을 박으면 | ~을 대입하면 |
| 풀어보면 | 풀면 |
| 식이 같다 | 등식이 성립한다 |

### 1-4. 단계별 서술 권장 패턴

1. **조건 분석** → 식 세우기
2. **식 정리** → 미지수 결정
3. **결과 계산**
4. (필요 시) 검산

연결어:
- "조건 (가)에서 ~이므로 ~이다"
- "근과 계수의 관계에 의하여 ~"
- "양변에 ~을 곱하면 ~"
- "따라서 ~이다 / 그러므로 ~이다"

### 1-5. 그림으로 답을 직접 읽지 말 것 ★ (graph SOT 규칙 8과 짝)

> ❌ "그래프에서 [a=2]임을 알 수 있다"
> ✅ "그래프의 x절편이 [1], [3]이므로 [f left(x right) =a left(x-1 right) left(x-3 right)]" (a 미지수 → 조건으로 결정)

학생이 그림 보고 때려 맞추는 풀이는 교육과정 위반.
**미지수 → 조건 → 결정 → 답** 흐름으로.

## 2. 학년·단원 공통 용어 통일

### 2-1. 함수의 그래프

| 사용 금지 | 올바른 표현 | 판정 기준 |
|---|---|---|
| 위로 열려 있다 / U자 모양 | **아래로 볼록** | 이차함수 [y=ax^2 +...] 에서 [a>0] |
| 아래로 열려 있다 / ∩자 모양 | **위로 볼록** | [a<0] |

**서술 예시**:
- "이차함수 [y=f left(x right)]의 그래프는 아래로 볼록이므로 최고차항의 계수는 양수이다."

### 2-2. 복소수

| 사용 금지 | 올바른 표현 |
|---|---|
| 실수부 | **실수부분** |
| 허수부 | **허수부분** |

복소수 [z=a+b i]에서 ([a], [b]는 실수)에 대해 "실수부분은 [a], 허수부분은 [b]" 라 쓴다.

### 2-3. 상용로그

| 사용 금지 | 올바른 표현 |
|---|---|
| 지표 | **정수부분** |
| 가수 | **소수부분** |
| log_e | **자연로그** ([ln] 표기) |

### 2-4. 함수의 증감·극값

| 용어 | 정의 |
|---|---|
| 증가한다 | [x_1 <x_2 ~~> f left(x_1 right) <f left(x_2 right)] |
| 감소한다 | [x_1 <x_2 ~~> f left(x_1 right) >f left(x_2 right)] |
| 극댓값 / 극솟값 | 극대/극소인 함숫값 |
| 극값 | 극댓값과 극솟값을 통틀어 부르는 말 |

## 3. 학년별 단원 범주 (점점 누적)

> **방침**: 각 단원에 들어가는 풀이 패턴·용어·표기를 발견할 때마다 본 절에 누적.
> 처음에는 비어 있어도 OK — 시간이 지나며 데이터가 쌓이는 구조.

### 3-1. 고1 — 공통수학1

#### 3-1-1. 다항식

- **인수분해**: 인수정리 → 조립제법 → 인수분해 흐름
- **나머지 정리**: [f left(x right)] 을 [x-a] 로 나눈 나머지 = [f left(a right)]
- **인수정리**: [f left(a right) =0] [LRARROW] [x-a] 가 [f left(x right)] 의 인수

#### 3-1-2. 방정식과 부등식

- **이차방정식 판별식**:
  - [D=b^2 -4ac]
  - [D>0]: 서로 다른 두 실근
  - [D=0]: 중근
  - [D<0]: 서로 다른 두 허근 (켤레복소수)
- **근과 계수의 관계 (이차)**: [ax^2 +bx+c=0] 의 두 근 [alpha], [beta] 에 대해
  - [alpha+beta=- {b} over {a}], [alphabeta= {c} over {a}]
  - (콤마로 구분된 식은 각각 별도 대괄호로)
- **근과 계수의 관계 (삼차)**: [ax^3 +bx^2 +cx+d=0] 의 세 근 [alpha], [beta], [gamma] 에 대해
  - [alpha+beta+gamma=- {b} over {a}]
  - [alphabeta+betagamma+gammaalpha= {c} over {a}]
  - [alphabetagamma=- {d} over {a}]
- **복소수 등호**: [a+b i=c+d i] [LRARROW] [a=c] 이고 [b=d]
- **이차함수 표준형**: [f left(x right) =ax^2 +bx+c=a left(x+ {b} over {2a} right) ^2 - {b^2 -4ac} over {4a}], 꼭짓점 [left(- {b} over {2a},~- {b^2 -4ac} over {4a} right)]

#### 3-1-3. 도형의 방정식

- *(누적 대기)*

#### 3-1-4. 집합과 명제

- 집합의 정의는 한 대괄호 안 (`[A= left{1,~2,~3 right}]`)
- 집합의 연산 변형은 등호 분리 (`[A cup B^C] [= left(A^C cap B right) ^C]`)
- 함의 [RARROW], 동치 [LRARROW]

#### 3-1-5. 함수와 그래프

- 함수 표기: `[f`:`X ~->~ Y]`
- 콜론 앞뒤 백틱(작은 공백), 화살표 앞뒤 틸더(큰 공백)

### 3-2. 고1 — 공통수학2

- *(누적 대기 — 도형의 방정식, 집합, 명제, 함수, 경우의 수 등)*

### 3-3. 고2 — 대수

- 지수·로그·삼각함수 표기 (한글 수식입력기 통합본 13장 참조)
- 자연로그는 `[ln`{x}]`

### 3-4. 고2 — 미적분I

- **극값 (미적분)**: 극댓값·극솟값 (사잇소리), 극값 = 극댓값+극솟값 통칭
- **수열의 극한**: 수렴/발산, 양의 무한대 [a_n ~~> inf], 음의 무한대 [a_n ~~> -inf]

### 3-5. 고2 — 확률과 통계

#### 3-5-1. 표기

| 용어 | 표기 |
|---|---|
| 확률 | `[rmP it left(A` right)]` |
| 조건부확률 | `[rmP it left(A`left|`B`right)]` |
| 기댓값 | `[rmE it left(X` right)]` |
| 분산 | `[rmV it left(X` right)]` |
| 표준편차 | `[sigma left(X right)]` |
| 정규분포 | `[rmN it left(m,~sigma^2 right)]` |
| 표준정규분포 | `[rmN it left(0,~1^2 right)]` |
| 표본평균 | `[bar{X`}]` |

#### 3-5-2. 용어

- ✅ 확률밀도함수, 확률질량함수, 기댓값(사잇소리)

#### 3-5-3. 조합·순열·중복조합 표기 ★

- 조합: `[{}_{n} C _{r}]` (수식 맨 앞일 때 `{}` 선행)
- 순열: `[{}_{n} P _{r}]`
- 중복조합: `[{}_{n} smallprod _{r}]` (★ `Pi` / `rmH` 사용 금지)
- 중복순열: `[{}_{n} smallprod _{r}]` (smallprod 가 Π — 한글 수식입력기 통합본 표기 표준)

## 4. 자주 쓰는 풀이 패턴 (전 학년)

### 4-1. 미지수 도입 표준 흐름 (그래프 문제)

| 그래프 | 미지수 | 결정 조건 |
|---|---|---|
| 이차함수 | [a left(a ne 0 right)] | x절편 + y절편 (또는 다른 한 점) |
| 일차함수 | [m left(m ne 0 right)], [n] | 두 점 |
| 삼각함수 | 진폭·주기 | 최댓값·주기 |
| 지수함수 | [a left(a>0,~a ne 1 right)] | 한 점 |

### 4-2. 이차함수 미지수 풀이 예시

> 그림에서 x절편 [1], [3], y절편 [3]인 이차함수
> 1. **미지수 도입**: [f left(x right) =a left(x-1 right) left(x-3 right)] ([a ne 0])
> 2. **조건 적용**: y절편이 [3] 이므로 [f left(0 right) =3a=3]
> 3. **a 결정**: [a=1]
> 4. **f(x) 결정**: [f left(x right) =left(x-1 right) left(x-3 right) =x^2 -4x+3]

### 4-3. 식 변형 시 등호 분리 (수식 입력기 통합본 2-2 참조)

```
[alpha^2 +beta^2][= left(alpha+beta right) ^2 -2 alphabeta][= {9} over {4} +8][= {41} over {4}]
```

방정식·판별식·근과 계수 관계는 한 대괄호 안에. 식 변형(등호 연속)은 분리.
