■ 서체 모드
  - 기본값: 이탤릭체 (italic)
  - rm       → 로마체(roman)로 전환
  - it       → 다시 이탤릭체로 복귀
  - 반드시 로마체(rm) 사용해야 하는 것:
      도형의 꼭짓점·점 (A, B, C …), 확률 P, 기댓값 E,
      분산 V, 정규분포 N, 순열 P, 조합 C, 중복조합 H

■ 공백
  - ` (백틱)  → 작은 띄어쓰기 (강제 공백 1칸)
  - ~         → 큰 띄어쓰기 (백틱 4개와 동일)
  - le, ge, ne 등 키워드 앞뒤에 다른 문자가 오면 반드시 공백 필요
      예) x ge 3   (O)    x ge3   (X)

■ 괄호
  - 소괄호  ( )  →  left( ... right)
  - 중괄호  { }  →  left{ ... right}    (코드상 {}와 구분)
  - 대괄호  [ ]  →  left[` ... `right]
  - 절댓값  | |  →  left|` ... `right|
  - 함수명과 left 사이 반드시 띄어쓰기
      예) f left(x right)   (O)    fleft(x right)   (X)

■ 기타 주의사항
  - 수식 맨 앞에 아래첨자가 올 때 → 빈 중괄호 선행
      예) {}_{n}{rmC it}_{r}
  - 로마체 대문자 알파벳으로 수식이 끝나거나 괄호가 오면 백틱 삽입
      예) rmP it left(A` right)
  - 수식 내 줄바꿈 → # (구간별 정의 함수에서만 사용)
  - 수식 내 한글이 필요한 경우 → 수식창과 한글창을 분리하여 입력

         x       (x>1)
