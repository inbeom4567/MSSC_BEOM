# 한글 수식 입력기 작성 규칙 (통합 완전판)

> **작성일**: 2026-04-30
> **목적**: 한글 수식 입력기(HWP 수식)로 수학 문제·해설을 작성할 때 적용하는 모든 규칙

---

## 1. 본문 작성 원칙 ⭐ (가장 중요)

### 1-1. 모든 숫자·알파벳은 대괄호로 감싼다

본문에 등장하는 **모든 숫자와 알파벳(변수)** 은 한글 수식 입력기 형식으로 대괄호 `[ ]`에 감싸서 출력한다. **예외 없음.**

| 잘못된 예시 | 올바른 예시 |
|---|---|
| `함수 f의 개수는 512` | `함수 [f]의 개수는 [512]` |
| `① 840` | `① [840]` |
| `1, 2, 3, 4 중에서 4개` | `[1, ~2,~ 3,~ 4] 중에서 [4]개` |

### 1-2. 변수 선언 구문은 각각 따로 대괄호 ⭐

"두 실수 a, b에 대하여" 등 **변수를 선언하는 구문**에서는 **각 변수를 별도의 대괄호로 분리**한다.

| 잘못된 예시 | 올바른 예시 |
|---|---|
| `두 실수 [a, b]에 대하여` | `두 실수 [a], [b]에 대하여` |
| `두 점 [rmA, rmB]` | `두 점 [rmA], [rmB]` |
| `두 근을 [alpha,~beta]` | `두 근을 [alpha], [beta]` |
| `두 상수 [M, m]에 대하여` | `두 상수 [M], [m]에 대하여` |

### 1-3. 콤마 분리 규칙 ⭐

수식 안에서 **콤마(,)로 구분된 여러 식**은 각각 별도의 대괄호로 분리한다.

| 잘못된 예시 | 올바른 예시 |
|---|---|
| `[alpha+beta=-a,~alphabeta=b]` | `[alpha+beta=-a], [alphabeta=b]` |
| `[a=0,~b=2]` | `[a=0], [b=2]` |
| `[M+m=1,~Mm=-1]` | `[M+m=1], [Mm=-1]` |

**콤마를 그대로 사용하는 경우 (예외):**
- 좌표 표기: `[left(a,~b right)]`
- 집합의 원소: `[left{1,~2,~3 right}]`
- 정규분포 모수: `[rmN it left(m,~sigma^{2} right)]`
- 단순 나열: `[1, ~2,~ 3,~ 4]`

### 1-4. 용어 통일 ⭐

| 사용 금지 | 사용할 표현 |
|---|---|
| 실수부 | **실수부분** |
| 허수부 | **허수부분** |
| 지표 (상용로그) | **정수부분** |
| 가수 (상용로그) | **소수부분** |
| log_e | **자연로그** (표기는 `ln`) |
| 위로 열린 (포물선) | **아래로 볼록** |
| 아래로 열린 (포물선) | **위로 볼록** |

### 1-5. 풀이 본문에 장식 기호 금지 ⭐

❌ 별표(`★`, `*`), 체크(`✓`), 강조 화살표(`→`) 등 장식 기호 사용 금지
✅ "따라서", "그러므로" 등 자연스러운 연결어로 결론 서술

## 2. 등호 분리 규칙 ⭐⭐ (매우 중요)

수식의 **종류와 상황**에 따라 등호(=) 앞에서 끊을지 말지가 다르다.

### 2-1. 끊지 않는 경우 (한 대괄호 안에 작성)

**(1) 집합의 정의/표기**
```
[A= left{1,~2,~3,~4 right}]
[B= left{x`|`x^{2} -3x+2=0 right}]
```

**(2) 방정식**
```
[2x^{2} +3x-8=0]
[x^{2} -6a-4=0]
```

**(3) 판별식**
```
[D=b^{2} -4ac]
[{D} over {4} = left(k-5 right) ^{2} -left(k-1 right) left(k-8 right)]
```

**(4) 근과 계수의 관계**
```
[alpha+beta=-a]
[alphabeta=b]
[alpha+beta=- {3} over {2}]
```

### 2-2. 끊는 경우 (등호 앞에서 별도 대괄호) ⭐

**(1) 함수의 정의·표기**
```
[f left(x right)][=x^{2} +2x-1]
[g left(x right)][= left(x-3 right) ^{2} +6]
[H left(x right)][=ax^{3} +bx^{2} +cx+d]
```

**(2) 식을 정리·변형하며 등호로 이어지는 경우 (항상 끊음, 집합 포함)**
```
[left(1+i right) ^{2}][=1+2 i+i^{2}][=2 i]

[A cup B^{C}][= left(A^{C} cap B right) ^{C}][= left(B-A right) ^{C}]

[alpha^{2} +beta^{2}][= left(alpha+beta right) ^{2} -2 alphabeta][= {9} over {4} +8][= {41} over {4}]
```

### 2-3. 방정식 정리 시 (특수)

방정식을 정리할 때는 **한 줄에 원래 식 전체 한 줄, 다음 줄에 정리한 식 전체 한 줄**로 분리하여 작성. 윗줄과 아랫줄을 등호 분리로 연결하지 않음.

```
[c left(x+2 right) ^{2} =-c left(x+3 right) left(x-4 right)]
[left(x+2 right) ^{2} =- left(x^{2} -x-12 right)]
[x^{2} +4x+4=-x^{2} +x+12]
[2x^{2} +3x-8=0]
```

각 줄이 좌변=우변 형태의 방정식이므로 **줄 안에서는 끊지 않고**, 줄 사이에 줄바꿈으로 정리 단계를 표현한다.

### 2-4. 빠른 판단

| 상황 | 끊는가? |
|---|---|
| 함수 정의 [f left(x right)] = ... | 끊음 |
| 방정식 (좌변=우변=0 형태) | 안 끊음 |
| 집합 정의 [A] = {...} | 안 끊음 |
| 판별식, 근과 계수의 관계 | 안 끊음 |
| 식 변형 (등호 두 개 이상 연속) | 끊음 |
| 집합 연산 변형 | 끊음 |

## 3. 그리스 문자 입력 ⭐

한글 수식 입력기에서 그리스 문자는 **영문 알파벳 이름**으로 입력해야 하며, 단순히 그 글자에 해당하는 알파벳을 쓰면 안 된다.

| 그리스 문자 | 입력 코드 | 영문 알파벳 사용 (잘못된 예) |
|---|---|---|
| α | `alpha` | a (X) |
| β | `beta` | b (X) |
| γ | `gamma` | c (X) |
| δ | `delta` | d (X) |
| ε | `epsilon` | e (X) |
| θ | `theta` | t (X) |
| λ | `lambda` | l (X) |
| μ | `mu` | m (X) |
| π | `pi` | p (X) |
| ρ | `rho` | r (X) |
| σ | `sigma` | s (X) |
| τ | `tau` | t (X) |
| φ | `phi` | f (X) |
| χ | `chi` | x (X) |
| ψ | `psi` | y (X) |
| **ω** | **`omega`** | **w (X)** ⭐ |

**특히 주의: ω(오메가)** ⭐
- ❌ `w` — 영문 알파벳 'w'로 인식됨
- ✅ `omega` — 그리스 문자 ω로 정상 출력

```
[omega^{3} =1]
[1+omega+omega^{2} =0]
```

대문자 그리스 문자: 첫 글자를 대문자로 (예: `Alpha`, `Theta`, `Lambda`, `Pi`, `Sigma`, `Phi`, `Psi`, `Omega`)
단, 대문자 Σ(시그마, 합)은 `sum`을, Π(파이, 곱)은 `prod`를 사용한다.

## 4. 사용자 정의 이항연산 별표 ⭐

사용자 정의 이항연산 별표는 **유니코드 ∗ (U+2217)** 를 직접 입력한다.

```
[A∗B=2A-B^{2}]
```

**복사용**: ∗

❌ 키보드 별표 `*` (U+002A) — 키보드 위쪽 작은 별표로 인식됨
❌ `ast` 등 명령어 사용 X
✅ 유니코드 ∗ 직접 입력

**예시 (등호 분리 규칙 적용)**:
```
[left(x^{2} -xy+2y^{2} right) ∗ left(2x-y right)]
[=2 left(x^{2} -xy+2y^{2} right) -left(2x-y right) ^{2}]
[=-2x^{2} +2xy+3y^{2}]
```

## 5. 각도 표시 ⭐

각도 표시 기호는 **반드시 대문자 `DEG`** 사용. 소문자 `deg`는 인식되지 않음.

```
[60 DEG]
[180 DEG -theta]
[rm{angle{ABC}} =it 90 DEG]
[sin`{30 DEG} = {1} over {2}]
```

❌ `deg` 사용 금지
✅ `DEG` 사용

## 6. 함수·관계 표기

### 6-1. 함수 표기 (정의역·공역)

```
[f`:`X ~->~ Y]
```
- 콜론 앞뒤: 백틱(`` ` ``) — 작은 공백
- 화살표 앞뒤: 틸더(`~`) — 큰 공백

### 6-2. 구간별 정의 함수 (cases)

```
[f left(x right) =cases{` 식1 && left(조건1 right) #` 식2 && left(조건2 right) }]
```

**예시:**
```
[f left(x right) =cases{` x^{2} +2x+1 && left(x le 1 right) #` x && left(x>1 right) }]
```

## 7. 폰트 (로마체·이탤릭체) 처리

### 7-1. rm 블록 후 it 복귀

`rm{...}` 로마체 블록 뒤에 이탤릭체 변수/수식이 이어지면 반드시 중간에 `it`을 넣어 폰트를 복귀시킬 것.

```
[rm{bar{AB}} =it k^{2} +2]
[rm{angle{AOC}} =it theta]
[rm{angle{ADB}} =it 90 DEG]
```

### 7-2. 로마체 사용 대상

- **확률·통계**: `rmP`, `rmE`, `rmV`, `rmN`
- **도형의 점**: A, B, C, D, O 등 (예: `rm{bar{AB}}`)
- **순열·조합·중복조합**: P, C, smallprod (이미 자동 로마체)

로마체 대문자 뒤에 괄호가 오면 **백틱(`) 삽입 필수**:
```
[rmP it left(A` right)]
[rmE it left(X` right)]
[rmN it left(m,~sigma^{2} right)]
```

---

## 8. 조합·순열·중복조합

### 8-1. 중복조합 기호: `smallprod` ⭐

❌ Pi 사용 금지
❌ rmH 사용 금지
✅ **`smallprod`** 만 사용

### 8-2. 맨 앞 빈 중괄호 `{}` ⭐

조합(C), 중복조합(smallprod), 순열(P) 기호가 **수식 맨 앞**에 올 때에는 반드시 앞에 빈 중괄호 `{}`를 붙일 것.

| 위치 | 코드 |
|---|---|
| 맨 앞 (조합) | `[{}_{6} C _{2} =15]` |
| 맨 앞 (순열) | `[{}_{6} P _{3} =120]` |
| 맨 앞 (중복조합) | `[{}_{4} smallprod _{4}]` |
| 중간 | `[2 times _{4} smallprod _{4}]` |

## 9. 말줄임표 (위치별 다름) ⭐

### 9-1. 본문 내 단독 말줄임 → `[cdotscdots]`

식과 식 사이, 변수 나열 사이에서 단독으로 사용될 때.

```
"숫자 [1, ~2,~ 3,~ 4,~ 5]가 적힌"
"[a_{1}], [a_{2}], [cdotscdots], [a_{10}]"
"[k=0], [1], [cdotscdots], [37]"
```

### 9-2. 수식·등식 내부 중간 생략 → `` ` cdots ` ``

한 식 안에서 항을 생략할 때는 백틱-공백-cdots-공백-백틱 형식.

```
[a_{10} x^{10} +a_{9} x^{9} +` cdots `+a_{1} x+a_{0}]
[1+2+3+` cdots `+n]
```

❌ 절대 등식 안에 `cdotscdots` 사용 금지

## 10. 절댓값·대괄호·가우스 기호

**안쪽에 백틱 사용 금지**, 일반 공백으로 작성.

| 종류 | 코드 |
|---|---|
| 절댓값 | `[left|z right|]` |
| 절댓값 (제곱) | `[left|z^{5} right|^{2}]` |
| 절댓값 (함수) | `[left|f left(x right) right|]` |
| 대괄호 | `[left[x+1 right]]` |
| 가우스 | `[left[x right]]` |

**예외**: 조건부확률처럼 `|`가 분리바 역할일 때만 백틱 사용
```
[rmP it left(A`left|`B`right)]
```

## 11. 부호·연산자

### 11-1. ± 기호 ⭐

❌ `pm` 사용 금지 (한글에서 인식 안 됨)
✅ **`+-`** 사용

```
[x= {-b +- sqrt{b^{2} -4ac}} over {2a}]
[a= {1 +- sqrt{5}} over {2}]
```

∓ 기호: `-+`

### 11-2. 부등호

`le`, `ge`, `ne` 앞뒤에 공백 필수 (분리바 역할의 `|` 제외).

```
[x le 5]    [a ne 0]    [b ge 0]
```

### 11-3. 허수단위 i 띄어쓰기 ⭐

허수단위 `i`가 다른 변수 뒤에 올 때 **반드시 한 칸 띄어 쓰기**.

| 잘못된 예시 | 올바른 예시 |
|---|---|
| `a+bi` | `[a+b i]` |
| `m+ni` | `[m+n i]` |
| `2abi` | `[2ab i]` |
| `4a^{2} bi` | `[4a^{2} b i]` |

## 13. 로그·삼각함수

함수명과 인수 사이에 **백틱(`) 필수**.

### 13-1. 로그
```
[log`{x}]               ← 상용로그 (밑 10)
[log_{3}`{x}]           ← 밑이 3인 로그
[log_{3}`{left(2x+1 right)}]
[ln`{x}]                ← 자연로그 (밑 e)
[ln`{e^{2}} =2]
```

### 13-2. 삼각함수
```
[sin`x]
[cos^{2} `{theta}]
[tan`{{pi} over {4}}]
[sin`{30 DEG} = {1} over {2}]
```

## 부록: 자주 쓰는 패턴 모음

### 복소수 관련
```
[z=a+b i]              ← 복소수 (방정식 형태, 안 끊음)
[bar{z`}]              ← 켤레복소수
[i^{2} =-1]            ← 항등식, 안 끊음
[omega^{3} =1]
[1+omega+omega^{2} =0]
```

### 이차방정식 (방정식이므로 안 끊음)
```
[ax^{2} +bx+c=0]
[x= {-b +- sqrt{b^{2} -4ac}} over {2a}]
[alpha+beta=- {b} over {a}]
[alphabeta= {c} over {a}]
[D=b^{2} -4ac]
```

### 식 변형 (등호 연속이므로 끊음)
```
[left(1+i right) ^{2}][=1+2 i+i^{2}][=2 i]

[alpha^{2} +beta^{2}][= left(alpha+beta right) ^{2} -2 alphabeta][= {9} over {4} +8][= {41} over {4}]
```

### 함수의 정의 (함수 정의이므로 끊음)
```
[f left(x right)][= left(x-1 right) ^{2} +p]
[g left(7 right)][= left(7-3 right) ^{2} +6][=22]
```

### 집합 (정의는 안 끊음, 연산 변형은 끊음)
```
[A= left{1,~2,~3,~4 right}]                     ← 안 끊음

[A cup B^{C}][= left(A^{C} cap B right) ^{C}]   ← 변형이므로 끊음
```

### 도형·각도
```
[rm{angle{ADB}} =it 90 DEG]
[rm{bar{AB}} =it 2a]
[sin`{60 DEG} = {sqrt{3}} over {2}]
```

### 사용자 정의 이항연산
```
[A∗B=2A-B^{2}]                                   ← 정의 (방정식형, 안 끊음)
[left(x^{2} +y^{2} right) ∗ left(x-y right)]    ← 식 시작
[=2 left(x^{2} +y^{2} right) -left(x-y right) ^{2}]   ← 변형이므로 끊음
```
