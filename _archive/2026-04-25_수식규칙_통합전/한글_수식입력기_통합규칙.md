# 한글 수식입력기 통합 규칙 문서

> **적용 범위**: 고등학교 수학 전 범위 / 한글(HWP) 프로그램 수식편집기 기준
> **목적**: 수학 문제·해설·교재 제작 시 일관된 수식 입력 형식 유지

---

## 0. 본문 작성 원칙 (가장 중요)

### 모든 숫자·알파벳은 대괄호로 감싼다

본문에 등장하는 **모든 숫자와 알파벳(변수)** 은 한글 수식 입력기 형식으로 대괄호 `[ ]` 에 감싸서 출력한다. **예외 없음.**

| 잘못된 예시 | 올바른 예시 |
|---|---|
| `함수 f의 개수는 512` | `함수 [f]의 개수는 [512]` |
| `① 840` | `① [840]` |
| `1, 2, 3, 4 중에서 4개` | `[1, ~2,~ 3,~ 4] 중에서 [4]개` |
| `숫자 1, 2, 3, 4, 5가 적힌` | `숫자 [1, ~2,~ 3,~ 4,~ 5]가 적힌` |

**쉼표로 나열된 숫자/문자열도 하나의 대괄호로 묶을 것.**

## 1. 전역 규칙

### 1-1. 서체 모드

- **기본값**: 이탤릭체 (italic)
- `rm` → 로마체(roman)로 전환
- `it` → 다시 이탤릭체로 복귀
- **반드시 로마체(rm)** 사용해야 하는 것:
  - 도형의 꼭짓점·점 (A, B, C ...)
  - 확률 P, 기댓값 E, 분산 V, 정규분포 N
  - 순열 P, 조합 C, 중복조합 H

> **중요**: `rm{...}` 로마체 블록 뒤에 이탤릭체 변수/수식이 이어지면 반드시 중간에 `it`을 넣어 폰트를 복귀시킬 것.
> 예: `[rm{bar{AB}} =it k^{2} -left(-2 right) =k^{2} +2]`

### 1-2. 공백

| 기호 | 의미 |
|---|---|
| `` ` `` (백틱) | 작은 띄어쓰기 (강제 공백 1칸) |
| `~` (틸더) | 큰 띄어쓰기 (백틱 4개와 동일) |

- `le`, `ge`, `ne` 등 키워드 앞뒤에 다른 문자가 오면 **반드시 공백 필요**
  - ✅ `x ge 3`
  - ❌ `x ge3`

### 1-3. 괄호

| 종류 | 코드 |
|---|---|
| 소괄호 ( ) | `left( ... right)` |
| 중괄호 { } | `left{ ... right}` |
| 대괄호 [ ] | `left[`` ... ``right]` |
| 절댓값 \| \| | `left|`` ... ``right|` |

- 함수명과 `left` 사이 **반드시 띄어쓰기**
  - ✅ `f left(x right)`
  - ❌ `fleft(x right)`

### 1-4. 기타 주의사항

- 수식 맨 앞에 아래첨자가 올 때 → **빈 중괄호 `{}` 선행**
  - 예: `[{}_{n} rmC it _{r}]`
- 로마체 대문자 알파벳으로 수식이 끝나거나 괄호가 오면 **백틱 삽입**
  - 예: `[rmP it left(A` right)]`
- 수식 내 줄바꿈 → `#` (구간별 정의 함수에서만 사용)
- 말줄임표(······)도 반드시 대괄호로 감쌀 것: `[cdotscdots]`

## 5. 함수

### 5-1. 함수 표기

| 수학 표기 | 한글 수식 코드 |
|---|---|
| f: X → Y | `[f`:`X ~->~ Y]` |
| f(x) | `[f left(x right)]` |
| f(g(x)) (합성함수) | `[f left(g left(x right) right)]` |
| f⁻¹(x) (역함수) | `[f^{`-1} left(x right)]` |
| f'(x) (도함수) | `[f`' left(x right)]` |
| f''(x) (이계도함수) | `[f`'' left(x right)]` |

### 5-2. 삼각함수 (함수명 뒤 백틱 필수)

| 수학 표기 | 한글 수식 코드 |
|---|---|
| sin x | `[sin`x]` |
| cos²θ | `[cos^{2} `{theta}]` |
| tan(π/4) | `[tan`{{pi} over {4}}]` |

### 5-3. 특수 표기

| 수학 표기 | 한글 수식 코드 |
|---|---|
| [x] (가우스) | `[left[`x`right]]` |
| \|f(x)\| (절댓값) | `[left|`f left(x right) `right|]` |

### 5-4. 구간별 정의 함수 (cases)

**형식**: `cases{` 식1 && 조건1 #` 식2 && 조건2 }`

- 각 식 앞에 **반드시 백틱(`)** 붙임
- 식과 조건 사이는 `&&`
- 케이스 구분은 `#` (앞뒤 공백 없음, 다음 케이스 시작은 백틱)

**예시**:
```
[f left(x right) =cases{` x^{2} +2x+1 && left(x le 1 right) #` x && left(x>1 right) }]
```

해석:
$$f(x) = \begin{cases} x^2 + 2x + 1 & (x \le 1) \\ x & (x > 1) \end{cases}$$

## 10. 확률·통계

### 10-1. 순열·조합 (수식 맨 앞에 올 때 `{}` 선행)

| 수학 표기 | 코드 (중간) | 코드 (맨 앞) |
|---|---|---|
| n! | `[n!]` | `[n!]` |
| ₙPᵣ | `[_{n} P _{r}]` | `[{}_{n} P _{r}]` |
| ₙCᵣ | `[_{n} C _{r}]` | `[{}_{n} C _{r}]` |
| ₙHᵣ (중복조합) | `[_{n} smallprod _{r}]` | `[{}_{n} smallprod _{r}]` |

> **중요**: 중복조합 기호는 `Pi`가 아니라 **`smallprod`** 사용
> 예시 (맨 앞): `[{}_{4} smallprod _{4} - _{3} smallprod _{4} =256-81=175]`
> 예시 (맨 앞): `[{}_{6} P _{3} =120]`, `[{}_{6} C _{2} =15]`

### 10-2. 확률 (로마체 P, 뒤에 백틱)

| 수학 표기 | 한글 수식 코드 |
|---|---|
| P(A) | `[rmP it left(A` right)]` |
| P(A∩B) | `[rmP it left(A cap B` right)]` |
| P(A\|B) (조건부 확률) | `[rmP it left(A`left|`B`right)]` |
| P(Aᶜ) | `[rmP it left(A^{C} right)]` |

### 10-3. 통계

| 수학 표기 | 한글 수식 코드 |
|---|---|
| x̄ (평균) | `[bar{X`}]` |
| σ² | `[sigma^{2}]` |
| σ | `[sigma]` |
| E(X) | `[rmE it left(X` right)]` |
| V(X) | `[rmV it left(X` right)]` |
| N(m, σ²) (정규분포) | `[rmN it left(m,~sigma^{2} right)]` |
| P(Z≤z) | `[rmP it left(Z le z right)]` |
