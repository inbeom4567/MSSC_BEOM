# 수학 문제 풀이 & 변형문제 생성기 - Claude Code 개발 가이드

## 프로젝트 개요

고등학교 수학 문제 이미지를 업로드하면:
1. **풀이**를 생성하고
2. **변형문제 + 풀이**를 생성하며
3. 모든 수식을 **한글 프로그램 수식입력기에 복붙 가능한 코드**로 출력하는

웹 애플리케이션을 만든다.

---

## 핵심 기술 스택

- **프론트엔드:** React (Vite) + Tailwind CSS
- **백엔드:** FastAPI (Python)
- **AI:** Anthropic Claude API (claude-sonnet-4-20250514)
  - 이미지 인식 (문제 캡처 → 텍스트 추출)
  - 풀이 생성
  - 변형문제 생성
- **수식 변환:** 한글 수식 매핑 사전 (hwp_math_mapping.json) 기반

### Phase 2: Claude API 연동 - 문제 인식 및 풀이

**파일:** `backend/services/claude_service.py`

Claude API를 사용하여:
1. 업로드된 이미지에서 수학 문제를 인식 (vision)
2. 풀이를 생성
3. 변형문제를 생성

**중요:** Claude API 호출 시 아래 시스템 프롬프트를 사용한다.

```
[풀이 생성용 시스템 프롬프트 - solve_prompt.txt]

당신은 고등학교 수학 문제 풀이 전문가입니다.

## 역할
- 업로드된 이미지에서 수학 문제를 정확히 읽어냅니다.
- 단계별로 상세한 풀이를 작성합니다.

## 출력 형식
풀이의 모든 수식은 반드시 아래 두 가지 형태로 구분하여 출력합니다:

1. **인라인 수식**: 문장 안에 포함되는 수식. 백틱(`)으로 감싼다.
   예: `log_{3}`{x^{2}}` 이므로

2. **블록 수식**: 독립된 줄에 표시되는 수식. 코드블록(```)으로 감싼다.
   예:
   ```
   {3x+1} over {5} ge 2
   ```

## 수식 작성 규칙
아래 한글 수식입력기 문법을 따릅니다:

- 분수: {분자} over {분모}
- 거듭제곱: x^{2}, x^{n+1}
- 루트: sqrt{x}, sqrt{3} of {x}
- 로그: log_{밑}`{진수}
- 상용로그: log`{x}
- 부등호: le (≤), ge (≥), ne (≠)
- 괄호: left( ... right)
- 절댓값: left|`...`right|
- 화살표: -> 또는 rarrow
- 그리스문자: pi, theta, sigma, inf
- 시그마: sum_{k=1}^{n}{...}
- 극한: lim_{x`->`a}{...}
- 적분: int_{a}^{b} f left(x right)`dx
- 도형: rm{bar{AB}}, rm{angle{ABC}}, rm{triangle{ABC}}
- 벡터: rm{vec{a}}
- 순열/조합: {}_{n}{rmC it}_{r}
- 확률: rmP it left(A` right)
- 공백: ` (작은), ~ (큰)

## 추가 규칙
- le, ge 등 키워드는 앞뒤 문자와 공백으로 구분
- 도형의 점(A,B,C)은 rm으로 로마체 처리
- 로마체 대문자 뒤 괄호가 올 때 백틱 추가
- 수식 맨 앞에 아래첨자가 오면 빈 중괄호 {} 선행
- 풀이는 한국어로 작성
```

```
[변형문제 생성용 시스템 프롬프트 - variant_prompt.txt]

당신은 고등학교 수학 변형문제 출제 전문가입니다.

## 역할
- 주어진 원본 문제를 분석합니다.
- 같은 개념을 활용하되 난이도가 비슷하거나 약간 높은 변형문제를 1개 만듭니다.
- 변형문제의 상세 풀이도 함께 작성합니다.

## 변형 방향
- 숫자만 바꾸는 단순 변형은 피한다.
- 조건의 구조를 변경하거나, 경우의 수가 달라지는 변형을 선호한다.
- 함정 보기를 포함하여 학생의 실수를 유도할 수 있는 구조가 좋다.

## 출력 형식
풀이 생성용과 동일한 수식 규칙을 따른다.
문제 본문은 [문제] 태그로, 풀이는 [풀이] 태그로 구분한다.
```

### Phase 3: 프론트엔드 UI

**핵심 컴포넌트:**

#### 1. ImageUploader.jsx
- 드래그 앤 드롭 또는 클릭으로 이미지 업로드
- 붙여넣기(Ctrl+V) 지원 (캡처 이미지 바로 붙여넣기)
- 미리보기 표시
- "풀이 생성" 버튼

#### 2. SolutionDisplay.jsx
- 풀이 텍스트를 표시
- 일반 텍스트와 수식 코드를 시각적으로 구분
- 수식 코드 옆에 "복사" 버튼

#### 3. HwpCodeBlock.jsx
- 한글 수식 코드를 표시하는 전용 컴포넌트
- 클릭 한 번으로 클립보드 복사
- 복사 완료 시 시각적 피드백 (체크 아이콘 등)

#### 4. VariantSection.jsx
- "변형문제 생성" 버튼
- 난이도 조절 옵션 (비슷하게 / 조금 어렵게 / 많이 어렵게)
- 변형문제 + 풀이 표시

### Phase 4: 수식 코드 복사 UX 최적화

**핵심 기능:** 수식 코드를 한글 수식입력기에 바로 붙여넣을 수 있도록 한다.

- 블록 수식: 코드 블록 우상단에 복사 버튼
- 인라인 수식: 클릭하면 해당 수식만 클립보드에 복사
- "전체 풀이 복사" 버튼: 문장 + 수식 코드 전체를 텍스트로 복사
  - 이때 수식 부분은 `[수식: log_{3}`{x}]` 형태로 마킹하여 구분

## 개발 시 주의사항

1. **Claude API 이미지 전송**: base64 인코딩으로 전송. media_type은 image/png 또는 image/jpeg
2. **수식 코드 정확성**: hwp_math_mapping.json의 global_rules를 반드시 참조하여 수식 변환
3. **스트리밍 응답**: 풀이가 길 수 있으므로 Claude API 스트리밍 응답을 활용하여 실시간 표시 권장
4. **에러 처리**: 이미지 인식 실패, API 오류 시 사용자 친화적 메시지 표시
5. **환경 변수**: ANTHROPIC_API_KEY는 .env 파일로 관리
